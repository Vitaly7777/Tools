# Преобразователь путей Samba

Простое приложение PyQt5 для преобразования путей Samba между различными форматами.

## Описание

Это приложение предоставляет удобный интерфейс для преобразования путей к сетевым ресурсам между различными форматами, часто используемыми при работе с общими ресурсами Samba. Оно позволяет преобразовывать пути:

*   Из формата Python с экранированием символов в формат UNC (Universal Naming Convention).
*   Из формата UNC в формат Python с экранированием символов.
*   Из формата UNC в формат SMB (Server Message Block).
*   Из формата SMB в формат UNC.

## Функциональные возможности

*   **Преобразование путей**: Простое преобразование путей между различными форматами с помощью понятного и интуитивно понятного графического интерфейса.
*   **Выбор типа преобразования**: Выберите желаемый тип преобразования с помощью переключателей (radio buttons).
*   **Очистка полей ввода и вывода**: Очистите поля ввода и вывода с помощью кнопки "Очистить".
*   **Копирование в буфер обмена**: Скопируйте преобразованный путь в буфер обмена с помощью кнопки "Копировать результат".
*   **Обработка ошибок**: Отображает предупреждающие сообщения для недопустимых входных путей или отсутствия направления преобразования.
*   **Подсказки**: Предоставляет полезные всплывающие подсказки для объяснения назначения каждой кнопки и опции.
*   **Горячие клавиши**: Удобные горячие клавиши для общих действий, таких как очистка, преобразование и копирование.
    *   **Ctrl+D**: Очистить поля ввода и вывода.
    *   **Enter**: Преобразовать путь.
    *   **Ctrl+C**: Скопировать результат в буфер обмена.

##  Скомпилированный файл под ОС Linux
*   В каталоге `./dist` находится скомпилированный файл `path_converter` под ОС Linux.
*   Для запука кликнети двойным щелчком по нему


## Требования для запуска файла `path_converter.py` (код на языке Python)

*   Python 3.x
*   PyQt5

### Установка

1.  Убедитесь, что у вас установлен Python 3.x.

2.  Установите PyQt5:
    Для Linux
*   pip install PyQt5==5.14.1
    Для Windows
*   pip install PyQt5==5.15.6

## Использование

1.  Запустите скрипт:
*   **Для запуска исходного кода:** `python path_converter.py`
*   **Для запуска скомпилированного файла (Linux):** Кликните двойным щелчком по файлу `path_converter` в каталоге `dist`.

2.  Выберите желаемый тип преобразования, используя переключатели.

3.  Введите путь, который вы хотите преобразовать, в поле "Введите путь:".

4.  Нажмите кнопку "Преобразовать", чтобы выполнить преобразование.

5.  Преобразованный путь отобразится в поле "Результат:".

6.  Нажмите кнопку "Копировать результат", чтобы скопировать преобразованный путь в буфер обмена.

## Интерфейс

Приложение предоставляет свои возможности через следующие основные элементы управления:

1.  **Тип преобразования пути**: отмечая интересующий вариант среди:
    *   _Убрать двойные \\\\ (Из Python в UNC)_
    *   _Добавить двойные \\\\ (Из UNC в Python)_
    *   _UNC в SMB (Из Windows в smb:)_
    *   _SMB в UNC (Из smb: в Windows)_
2.  **Путь к преобразованию**: его необходимо ввести в текстовом поле под названием _Введите путь_, чтобы позже, с помощью элемента управления _Преобразовать_, передать его в желаемом формате пути.
3.  **Выходной путь**: результаты самого преобразования отображаются в панели _Результат_, где, в свою очередь, есть элементы управления, такие как _Очистить_, _Преобразовать_ и _Копировать результат_, чтобы выполнять шаги по очистке, преобразованию и копированию информации.



## Пример

1.  Выберите "UNC в SMB (Из Windows в smb:)".
2.  Введите `\\server\share` в поле "Введите путь:".
3.  Нажмите "Преобразовать".
4.  В поле "Результат:" отобразится `smb://server/share`.
5.  Нажмите "Копировать результат", чтобы скопировать `smb://server/share` в буфер обмена.
