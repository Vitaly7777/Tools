# cut_files copy.py - Скрипт обрезки Excel файлов xlsb/xlsx/xlsm файлов

## Назначение

Этот скрипт предназначен для обработки файлов электронных таблиц с расширениями .xlsb, .xlsx, .xlsm и .ods. 
Он обрезает каждый лист в файле до указанного количества строк (по умолчанию 50 строк) и 
сохраняет результаты в новую папку, сохраняя исходную структуру подпапок. 
Это полезно для уменьшения размера файлов для тестирования или анализа, без потери начальных данных.
Если указанное количество строк равно 0 или не является целым числом, то файлы не обрезаются и 
сохраняются ТОЛЬКО значения ячеек  без формул и форматирования в формате *.xlsx.

Основные особенности:
Обрабатывает все листы в файле.
Сохраняет обрезанные файлы в формате .xlsx с сохранением структуры папок.
Выводит прогресс обработки и логирует ошибки.
Работает только с папкой data в рабочей директории.

## Структура папки cutter

```
excel_cut/
├── cut_files.py      # Скрипт обрезки
├── requirements.txt   # Зависимости Python
└── README.txt         # Этот файл
└── data/              # Каталог с файлами Excel(может содержать подкаталоги или быть переименован) 
└── data_сur/          # Каталог с выходными файлами Excel в формате .xlsx(опционно, если его нет скрипт создаст его) 
```

## Установка зависимостей
Зависимости: Скрипт требует Python 3.x и следующие библиотеки:
   - pandas (для работы с Excel).
   - openpyxl (для чтения и записи Excel-файлов).
   - calamine (для обработки .xlsb-файлов; если не установлен, установите с помощью pip).
Установите зависимости командой:

```bash
pip install -r requirements.txt
```

## Подготовка

Клонирование или копирование: Поместите скрипт в вашу рабочую директорию. Файл скрипта должен называться cut_files.py.
Скопируйте папку с данными (файлами Excel) в директорию `data` относительно скрипта. 
   Например:
  - Если скрипт находится в ./cut_files.py, то данные должны быть в `./data`.
или
  - Отредактируйте `cut_files.py`, если папка с данными отличается от `./data`:

`python code:`
Замените строку в коде:
   forwork_dir = WORK_DIR / 'data'
на:
   forwork_dir = WORK_DIR / '/ваша/рабочая/директория'


## Использование

Запустите скрипт:

```bash
python cut_files.py
```

## Что делает скрипт

 Проверит наличие папки data.
   - Создаст новую папку с суффиксом _cut (например, data_cut) в родительской директории.
   - Обработает все файлы .xlsb, .xlsx и .xlsm в data и её подпапках.
   - Сохранит обрезанные файлы в новой структуре (с сохранением подпапок).
Пример:

Если у вас есть файл ./data/АБС/ВАС/file.xlsb, обрезанный файл будет сохранён в ./data_cut/АБС/ВАС/file.xlsx.

Конфигурация
Скрипт использует глобальные переменные для настройки:

ROWS_TO_KEEP = 50: Количество строк для сохранения на каждом листе. Измените это значение в коде, если нужно.
  - Если указанное количество строк равно 0 (или не является целым), то файлы не обрезаются и 
   сохраняются ТОЛЬКО значения ячеек без формул и форматирования в формате *.xlsx.
forwork_dir = WORK_DIR / 'data': Папка для сканирования. Если вы хотите изменить, обновите эту строку.



## Важные моменты

- Все файлы сохраняются как **xlsx**, даже если исходный формат был **xlsb** или **xlsm**
- Файлы для отправки находятся в папках с суффиксом `_cut`
- Исходные файлы НЕ изменяются
- Скрипт показывает прогресс: `[1/10] file.xlsb`
